# 🌸 Bloomia｜跨境美妝電商網站

> **Bloom Your Confidence**  
> 綻放妳的自信

一個以「越南女性市場」為目標的跨境美妝電商前端專案，結合品牌策略思維與現代前端工程化開發。

---

## 🎯 專案定位（商業思維）

本專案並非單純切版練習，而是以真實商業情境進行設計：

- 🎯 目標市場：越南女性消費者
- 💄 商業模式：跨境美妝電商（台灣直郵）
- 🧴 品牌策略：
  - 建立「專業美容顧問」定位，而非純電商平台
  - 強化高信任感（國際代理商 / 沙龍級品牌）
  - 透過內容與教學提升轉換率

👉 核心思維：**不是賣產品，而是先建立信任，再促成購買**

---

## 🛠️ 技術架構與設計決策

### Frontend Stack

- ⚡ Vite（快速開發與 build）
- 🎨 Bootstrap 5（RWD Grid 系統）
- 🧠 JavaScript (ES6+)
- 🎨 SCSS（模組化樣式管理）
- 🧩 Swiper（輪播）
- 🔍 GLightbox（燈箱效果）

---

## 🧩 架構設計思維

### ✔️ 為什麼不使用 React / Vue？

- 本專案重點在 UI/UX 與切版能力展示
- 降低框架複雜度，專注在視覺與互動呈現
- 使用 Vite + Vanilla JS 保持控制彈性

---

### ✔️ Bootstrap + SCSS 並存策略

- Bootstrap：負責 RWD grid 與基礎 layout
- SCSS：負責品牌視覺與 design system

👉 設計策略：
> Bootstrap 做結構骨架，SCSS 做品牌層

---

## ✨ 技術亮點

### 1️⃣ 電商 UX Flow 設計

- 商品瀏覽 → hover → add to cart → toast feedback
- 即時操作回饋
- 模擬真實電商互動流程

---

### 2️⃣ Swiper + Card Layout 優化問題

**問題：**
- card 高度不一致造成 layout 跳動
- swiper 在 RWD 下壓縮變形

**解法：**
- 使用 flex 統一 card height
- image aspect ratio 控制比例
- 固定 card structure

---

### 3️⃣ RWD（Mobile First）策略

- 以 mobile 優先設計資訊層級
- desktop 再強化視覺與留白
- Bootstrap grid + custom breakpoints
